# csc125-final-project

# Project Description
This program is a password manager that allows users to store and manage account passwords in a simple and organized way. It is designed for anyone who wants a basic tool to keep track of multiple accounts without relying on external software.

# Features
- Dictionary storing {"gmail": "Abc123!", "netflix": "Sudo456@", "moodle": "Secure789#"}
- Custom function `[check_strength]` that checks if the password has at least one uppercase, lowercase, number, and special character. 
- while loop that repeatedly display a menu and allow the user to choose actions. This loop continues until the user selects the exit option
- User input with validation
- Formatted output
